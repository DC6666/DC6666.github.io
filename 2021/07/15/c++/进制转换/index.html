关于进制转换

进制转换在初高中就已经涉及，10进制转换为其他进制一般采用连续取余的方法，例如下面是将10进制转换为8进制的简单程序

例1

#include <iostream>
#include <stack>
#include <cmath>
using namespace std;
stack <int> s;
int main() {
	int n, base{8};//数字，base为基数
	//cin >> n >> base;
    cin>>n;
	bool is{ false };
	if (n < 0)is = true;
	while (n != 0) {
		s.push(abs(n % base));
		n /= base;
	}
	if (is)cout << "-";
	while (!s.empty()) {
		cout << s.top();
		s.pop();
	}
}
//当输入为15 8时，输出为17

其他进制的转换，如2进制转换为15进制，将其转换为10进制，在连续取余转换为15进制即可。

如：

例2

#include <iostream>
#include <cmath>
using namespace std;
int main() {
	int n, base;
	n = 1111; base = 2;//原数值和进制。
	int m = n;
	int x{ 0 }, w{ 0 };
	while (m != 0) {
		x += m % 10 * pow(base, w);
		m /= 10;
		w++;
	}
	cout << x;
    
}

之后将其用例1的方法转转换为其他15进制即可。

基数为负数的整数进制转换

针对于基数为正整数的整数的进制转换并不需要考虑太多因素，很如一就做出来，但是，当遇到基数为负数的进制转换在采用同样的方法明显就不适用了，例如在例1中，如果输入10 -2,结果将会是1010，结果是存在问题的。因此有了以下方法。

列3:

#include <iostream>
#include <stack>
#include <cmath>
using namespace std;
char c[]{ "0123456789ABCDEFGHIJK" };
stack <char> s;
int main() {
	int n, base;//数值，10进制，base为基数
	cin >> n >> base;
	bool is{ false };
	if (n < 0)is = true;
	while (n != 0) {
		int x = n % base;
		if (x < 0) {
			x = n - (n / base + 1) * base;
			n = n / base + 1;
		}
		else {
			n /= base;
		}
		s.push(c[x]);
	}
	while (!s.empty()) {
		cout << s.top();
		s.pop();
	}
}

这样可以解决基数为负数的情况。

例3是用来解决10进制转换为负数进制的问题的，如果是其他负数进制相互转换可以用例2的方法转换为10进制,然后再进行转换。
