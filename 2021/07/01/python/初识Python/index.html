类型转换

Python 提供了能够将值从一种类型转换为另一种类型的内建函数。函数 int 接受任意
值，并在其能做到的情况下，将该值转换成一个整型数，否则会报错：

 int ('32 ')
32
 int ('Hello ')
ValueError : invalid literal for int () : Hello

int 能将浮点数转换为整型数，但是它并不进行舍入；只是截掉了小数点部分：

 int (3.99999)
3
 int(−2.3)
−2

float 可以将整型数和字符串转换为浮点数:

 float (32)
32.0
 float ('3.14159 ')
3.14159

str 可以将其实参转换成字符

 str (32)
'32 '
 str (3.14159)
'3.14159 '

码赋予一个名字，这充当了某种文档说明。另一个好处是，如果你重复使用这些代码，

泛化

def polygon (t , n , length ) :
	angle = 360 / n  #python2 中可能会因为整型常量出现偏差，可以用360.0/n,有点像C语言哈哈
	for i in range (n ) :
		t . fd ( length )
		t . lt ( angle )

下面是一个画弧的程序

import math
import turtle

def arc (t,r,angle):
    arc_length = 2 * math . pi * r * angle / 360#周长
    n = int(arc_length / 3) + 1#三等分为n
    step_length = arc_length / n#每次作图的长度（路径）
    step_angle = angle / n#角度
    for i in range(n):#执行n次，前后呼应
        t . fd(step_length)
        t . lt(step_angle)
t = turtle.Turtle()
arc(t, 100, 270)#t是小乌龟，100是半径，270是弧度
turtle.done()
