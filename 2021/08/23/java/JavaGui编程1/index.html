JavaGui编程1

关于GUI编程

GUI全称是Graphical User Interface,即图形用户接口,就是用来提供给用户操作的图形界面的接口,通过这些提供的图形用户接口就可以实现包括窗口,菜单、按钮、工具栏和其他各种图形界面元素,从而实现程序的图形用户界面来方便用户操作。

Swing顶级容器

Swing 提供了三个主要的顶级容器类:JFrame,JDialog和JApplet,其中JFrame和JDialog 是最为常用也是最为简单的顶级容器。

JFrame容器

JFrame容器是一个独立的容器（窗口），不能放在其他容器中。支持窗口所有的基本功能。

public class GUI {
	private static void createGui() {
		//创建JFrame容器窗口
		JFrame frame = new JFrame("Test");
		//设置关闭窗口默认操作
		frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
		//设置窗口尺寸
		frame.setSize(520,520);
		//展示窗口
		frame.setVisible(true);
	}
	public static void main(String[] args) {
		//调用createGui方法显示窗口
		SwingUtilities.invokeLater(GUI::createGui);
	}
}

运行就会出现一个空白的窗口了。

JDialog容器

JDialog是Swing 的另外一个顶级容器,通常用来表示对话框窗口。JDialog对话框可分为两种:模态对话框和非模态对话框。所谓模态对话框是指用户需要等到处理完对话框后才能继续与其他窗口交互,而非模态对话框允许用户在处理对话框的同时与其他窗口交互。

下面再原本的JFrame容器的基础上加上JDialog容器

public class GUI {
	private static void createGui() {
		//创建JFrame容器窗口
		JFrame frame = new JFrame("Test");
		//设置关闭窗口默认操作
		frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
		//设置窗口尺寸
		frame.setSize(520,520);
		//展示窗口
		frame.setVisible(true);
		//在JFramer容器上方创建JDialog容器窗口，modal为true时，关闭JDialog后才可以操作JFrame
		JDialog dialog = new JDialog(frame,"JDialog对话框",false);
		dialog.setDefaultCloseOperation(JDialog.HIDE_ON_CLOSE);
		dialog.setSize(100,100);
		dialog.setVisible(true);
	}
	public static void main(String[] args) {
		//调用createGui方法显示窗口
		SwingUtilities.invokeLater(GUI::createGui);
	}
}

运行就会出现两个叠在一起的空白窗口
