在java中使用正则表达式、

什么是正则表达式

如何判断一个字符串是否是’10086’，在java中可以这样：

public class Test{
    public static void main(String[] args){
        String s = "abcde";
        System.out.println(s=="10086");//这里返回false
        String ss = "10086";
        System.out.println(ss.equals("10086"));//返回true
    }
}

那么怎么判断一个字符串是否是’10’开头’86’结尾，中间只有一个其他数字的字符串呢？

只要匹配到开头两个数字和末尾两个数字，判断中间只有一个数字就行，很容易通过编程实现这个目标，但是如果要匹配其他情况呢？编写很多程序是不可靠的，这时候就要使用<b>正则表达式</b>

回到刚才那个问题，如何判断一个字符串是否是’10’开头’86’结尾，中间只有一个其他数字的字符串。

我们使用“10\d86”来匹配”10086”,其中\d就代表’10’和’86’的中间可以有一个任意的数字，类似于一个宅占位符。

在<b>java.util.regex</b>中内置了正则表达式引擎。

import java.util.*;//java.util.regex
public class Test {
	public static void main(String[] args){
        String regex = "10\\d86";//对于java来书\是转义字符，所以用'\\'代表'\'
        System.out.println("10086".matches(regex));//输出true
        System.out.println("100086".matches(regex));//输出false
        System.out.println("10e86".matches(regex));//输出false
        
    }
}

正则表达式匹配规则

精准匹配

public static void main(String[] args){
    String regex = "10086";
    System.out.println("10586".matches(regex));//输出false
    System.out.println("10086".matches(regex));//输出true
    //如果有特殊字符就用'\'转义
    String regexs = "10086\"";
    System.out.println("10086\"".matches(regexs));
}

匹配任意字符.

public static void main(String[] args){
		String regex = "10.86";
        System.out.println("10086".matches(regex));//true
        System.out.println("10e86".matches(regex));//true
    }

匹配数字\d

‘\D’与之相反匹配非数字

public static void main(String[] args){
    String regex = "10\\d86";
    System.out.println("10086".matches(regex));//true
    System.out.println("10e86".matches(regex));//false
}

匹配常用字符\w

\w能匹配：字母、数字、’_’

不能匹配：!, {, }, [, ], ~, !, ！, @, #, $, %, ^, &, *, (, ), -, +, /, ?, ", |等

但是\W却与之相反

public static void main(String[] args){
    String regex = "10\\w86";
    System.out.println("10_86".matches(regex));//true
    System.out.println("10086".matches(regex));//true
    System.out.println("10d86".matches(regex));//true
    System.out.println("10=86".matches(regex));//false
    System.out.println("10#86".matches(regex));//false
    System.out.println("10 86".matches(regex));//false
}

匹配空格\s

‘\S’与其相反

public static void main(String[] args){
    String regex = "10\\s86";
    System.out.println("10 86".matches(regex));//true
}

重复匹配*

当匹配多个数字时可以：

public static void main(String[] args){
	String regex = "10\\d*86";//*匹配任意多字符
	System.out.println("10123456789086".matches(regex));//true
    String regex = "10\\d{3}86";//匹配3个数字
    System.out.println("1012386".matches(regex));//true
}

总结

正则表达式规则可以匹配
A指定字符A
\u548c指定Unicode字符和
.任意字符a，b，&，0
\d数字0~90~9
\w大小写字母，数字和下划线a~z，A~Z，0~9，_
\s空格、Tab键空格，Tab
\D非数字a，A，&，_，……
\W非\w&，@，中，……
\S非\sa，A，&，_，……
在文末菜鸟教程的链接中有更多其他字符。

另外：

\d+ \w+ \s+ 可以匹配一个或者多个数字，空格，字母,(.*) 或 .* 可以匹配多个任意字符

如以下结果全为 true：

public static void main(String[] args) {
    String regex1 = "\\d+";
    System.out.println("123456789".matches(regex1));
    String regex2 = "\\s+";
    System.out.println("       ".matches(regex2));
    String regex3 = "\\w+";
    System.out.println("abcdefg".matches(regex3));
    //当要使用+时就要使用转义字符
    String regex4 = "\\d\\+";
    System.out.println("1+".matches(regex4));
    //匹配.时要使用转义字符
    String regex5 = ".*\\.baidu\\..*";
    System.out.println("www.baidu.".matches(regex5));
    //System.out.println("www,baidu.".matches(regex5));//false
}

? 表示括号中的选项时可选的

如：

public static void main(String[] args) {
    String regex1 = "\\d+(\\w)?";
    System.out.println("123456789".matches(regex1));//true
    System.out.println("123456789s".matches(regex1));//true
}

matches 方法也可以这样使用

public static void main(String[] args) {
    String regex1 = "\\d+(\\w)?";
    System.out.println(Pattern.matches(regex1,"123456789"));//true
}

复杂的匹配规则

匹配开头结尾^,$

public static void main(String[] args){
	String regex = "^A\\d{3}$";//^表示开头，$表示结尾
    System.out.println("A234".matches(regex));//true               
}

匹配指定范围

1.用[]指定可以是哪些元素，用{}指定多少个或者第几个到第几个

[1-3] [1,2,3]代表1到3的数字.

注意:

{1,3}代表存在1到3个

{1,}代表存在至少1个

{3}代表有3个

public static void main(String[] args) {
    String regex1 = "\\d{2,3}";//表示有2到3个
    System.out.println("12".matches(regex1));//true
    System.out.println("123".matches(regex1));//true
    String regex2 = "\\d{3}";
    System.out.println("1234".matches(regex2));//true表示有3个
}

^和|

^:代表取反

public static void main(String[] args) {
    String regex1 = "[^1,2,3,4]\\d{1,3}";   //有2到4个除了1，2，3，4的数字
    System.out.println("5555".matches(regex1));//true
    String regex2 = "[^2-4]\\d{1,2}(.*)$";    //有2到3个除了2，3，4的数字，后面接着是任意个字符(.*)表示任意个任意字符，除了`\n`
    System.out.println("567abcd".matches(regex2));//true
}

|:代表或

public static void main(String[] args) {
    String regex1 = "[1,2,3,4]\\d{1,3}|abc";
    System.out.println("abc".matches(regex1));//true
    System.out.println("1234".matches(regex1));//true
    String regex2 = "[1-4]\\d{1,2}|[5-9]\\d{3,4}";
    System.out.println("123".matches(regex2));//true
    System.out.println("5678".matches(regex2));//true
}

使用()

以下两种正则表达式等效

String s="learn\sjava|learn\sphp|learn\sgo"
String s = "learn\\s(java|php|go)"

分组匹配

public static void main(String[] args) {
    String s = "(\\d{3,4})-(\\d{7,8})";
    System.out.println("010-12345678".matches(s));//true
    Pattern p = Pattern.compile("(\\d{3,4})-(\\d{7,8})");
    Matcher m = p.matcher("010-12345678");
    if (m.matches()) {
        System.out.println(m.group(0));//010-12345678
        System.out.println(m.group(1));//010
        System.out.println(m.group(2));//12345678
    } else {
        System.out.println("匹配失败!");
    }
}

基本方法

String[] split(CharSequence input);//围绕此模式的匹配拆分给定的输入序列。

使用Pattern 创建一个匹配此模式的给定输入的匹配器。

Matcher matcher(CharSequence input);

如：

Pattern pattern = Pattern.compile("java([0-5])");		Matcher matcher = pattern.matcher("java1.8isyyds");

Matcher

关于分组：

：组是用括号划分的正则表达式，可以根据组的编号来引用这个组。组号为0表示整个表达式，组号为1表示被第一对括号括起的组，依次类推，例如A(B(C))D，组0是ABCD，组1是BC，组2是C;如(AB)(CD)，组0时ABCD,组1是AB，组2是CD；

Pattern pattern();//返回此匹配器解释的模式 
boolean find();//尝试查找与模式匹配的输入序列的下一个子序列。  
boolean find(int start);//重置此匹配器，然后尝试从指定的索引处开始查找与模式匹配的输入序列的下一个子序列。
boolean lookingAt();//尝试将从区域开头开始的输入序列与模式匹配。
boolean matches();//尝试将整个区域与模式匹配。
int start();//返回上一个匹配的起始索引。  
int start(int group);//返回上一个匹配操作期间给定组捕获的子序列的起始索引。  
int end();//返回最后一个字符匹配后的偏移量。  
int end(int group);//返回在上一个匹配操作期间由给定组捕获的子序列的最后一个字符之后的偏移量。  
int end(String name);//返回在上一个匹配操作期间由给定 named-capturing group捕获的子序列的最后一个字符之后的偏移量;
String group();// 返回上一个匹配项匹配的输入子序列。  
String group(int group);// 返回上一个匹配操作期间给定组捕获的输入子序列。  
String group(String name);//返回在上一个匹配操作期间由给定 named-capturing group捕获的输入子序列。
int groupCount();// 返回此匹配器模式中捕获组的数量。

例如：

Pattern pattern = Pattern.compile("(jdk([0-9].?[0-9]))");
Matcher matcher = pattern.matcher("jdk1.8 is yyds,jdk11 is yyds too");
System.out.println(matcher.groupCount());//2 表示一共有2个组
Pattern pattern = Pattern.compile("(jdk([0-9].?[0-9]))");
Matcher matcher = pattern.matcher("jdk1.8 is yyds,jdk11 is yyds too");
System.out.println(matcher.find());//true   相当与find(0),创建一个匹配器从头匹配
System.out.println(matcher.group(1));//jdk1.8   返回第一组的匹配序列
System.out.println(matcher.start(1));//0   返回第一组的匹配序列的开始索引
System.out.println(matcher.end(1));//6    返回第一组的匹配序列的结束索引的后一个索引。
System.out.println(matcher.group(2));//1.8   返回第二组的匹配序列
System.out.println(matcher.start(2));//3
System.out.println(matcher.end(2));//6
System.out.println(matcher.find(2));//true   重置匹配器，将从下标为2的元素开始，创建一个新的匹配，这里也可以使用find(),相当于查找第三行代码查找序列的下一个子序列。
System.out.println(matcher.group(1));//jdk11   
System.out.println(matcher.start(1));//15
System.out.println(matcher.end(1));//20
System.out.println(matcher.group(2));//11
System.out.println(matcher.start(2));//18
System.out.println(matcher.end(2));//20

tips:每次进行group(),start(),end()操作前都要进行find()操作，也就是先创建一个匹配器。
