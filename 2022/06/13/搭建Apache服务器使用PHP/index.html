搭建Apache服务器使用PHP

安装Apache

下载

下载地址

也可以前往官网下载

下载Linux版本

wget https://dlcdn.apache.org/httpd/httpd-2.4.54.tar.gz

解压

tar zxvf httpd-2.4.54.tar.gz

安装

先安装apr

链接 

wget https://dlcdn.apache.org/apr/apr-1.7.0.tar.gz

解压apr

tar -zxvf apr-1.7.0.tar.gz

安装apr

/usr/local/apr/是安装目录

cd apr-1.7.0/
./configure --prefix=/usr/local/apr/
make
make install

安装apr-util

#下载
wget https://dlcdn.apache.org/apr/apr-util-1.6.1.tar.gz
#解压
tar -zxvf apr-util-1.6.1.tar.gz
cd apr-util-1.6.1
#安装可能缺少的依赖
yum install -y expat-devel
yum install -y libxml2-devel
#安装 /usr/local/apr-util/是安装目录 /usr/local/apr/是apr的安装目录
./configure --prefix=/usr/local/apr-util/ --with-apr=/usr/local/apr/
#安装
make
make install

安装apache

移动到先前解压的Apache目录中

cd ../httpd-2.4.54/

同安装apr一样安装

#加上apr和apr-util的安装路径
./configure --prefix=/usr/local/apache/ --with-apr=/usr/local/apr/ --with-apr-util=/usr/local/apr-util/
make
#安装时间比较久
make install

不出意外的话就安装成功了

安装PHP

下载解压安装PHP

#下载
wget https://www.php.net/distributions/php-8.0.20.tar.gz
tar -zxvf php-8.0.20.tar.gz
cd php-8.0.20/
./configure --prefix=/usr/local/php8 --disable-fileinfo
make && make install
#拷贝php.ini php.ini-development make install后的当前目录下
cp 'php.ini-development' php.ini /usr/local/php8/
cp php.ini /usr/local/php8/

如果出现报错 XXX not found 可以安装相关内容,如

yum -y install sqlite-devel

如果出现Package ‘oniguruma’, required by ‘virtual:world’, not found错误,执行一下代码(最好一行一行复制执行),到php安装包目录下执行 make distclean 在运行上一步的5-9行代码。

cd /usr/local/src/
wget https://github.com/kkos/oniguruma/archive/v6.9.4.tar.gz -O oniguruma-6.9.4.tar.gz
tar -zxvf oniguruma-6.9.4.tar.gz
yum install autoconf automake libtool #有询问输入y
cd oniguruma-6.9.4/
./autogen.sh && ./configure --prefix=/usr
make && make install

配置Apache

进入Apache安装目录下的conf目录,编辑 httpd.conf

vim /usr/local/apache2.4/conf/httpd.conf #编辑配置文件
#修改以下四点
#1.ServerName
ServerName www.example.com:80
#2.Directory 
<Directory />
    AllowOverride none
    Require all granted
</Directory>
#3.DirectoryIndex 
<IfModule dir_module>
    DirectoryIndex index.html index.php#主要是添加默认访问的文件，一般是index
</IfModule>
#4.AddType
AddType application/x-httpd-php .php

到Apache的bin目录下

cd ../bin
./apachectl -t #测试配置语法
./apachectl graceful #重新加载配置文件

运行

./apachectl start
curl localhost:8080 #测试

如果配置了域名可以通过域名加端口访问，http默认80端口，https默认443端口

停止

./apachectl stop
