MongoDB Java Driver

基本操作

ConnectionString

表示一个连接字符串。连接字符串描述了要使用的主机和选项。

mongodb://[username:password@]host1[:port1][,host2[:port2],...[,hostN[:portN]]][/[database.collection][?options]]
参数详情
  • mongodb:// : 是必需的前缀，以标识这是标准连接格式的字符串。
  • username:password@ : 用户名:密码:可选:@。如果给出，驱动程序将尝试登录到数据库服务器连接后的数据库。对于某些身份验证机制，只指定用户名而不指定密码，在这种情况下，用户名之后的“:”也会被删除
  • host1 : 是连接字符串中唯一必需的部分。它标识要连接的服务器地址。
  • :portX : 是可选的，如果没有提供，默认为:27017。
  • /database : /database是要登录的数据库名，因此只有在使用`username:password@`语法时才相关。如果不指定，默认使用“admin”数据库。
  • ?options : 连接选项。注意，如果数据库不在，在最后一个主机和`?`之间仍然需要 `/`。选项为“名称=值”对，用“&”分隔。为了向后兼容，";"可以作为除"&"之外的分隔符接受，但应视为已弃用。 
mongodb+srv://[username:password@]host[/[database][?options]]
参数详情
  • mongodb+srv://是此格式所需的前缀。
  • username:password@如果给出，驱动程序将尝试登录到数据库服务器连接后的数据库。对于某些身份验证机制，只指定用户名而不指定密码，在这种情况下，用户名之后的“:”也会被删除
  • host是URI中唯一必需的部分。它标识一个主机名，在主机名前面加上“_mongodb._tcp”，然后从域名服务器查找SRV记录。每个SRV记录的主机/端口成为用来连接的种子列表，就像使用正常的mongodb协议在URI中提供的每个主机/端口对一样。
  • /database/database是要登录的数据库名，因此只有在使用“用户名:密码@”语法时才相关。如果不指定，默认使用“admin”数据库。
  • ?options连接选项。注意，如果数据库不在，在最后一个主机和`?`之间仍然需要 `/`。选项为“名称=值”对，用“&”分隔。为了向后兼容，";"可以作为除"&"之外的分隔符接受，但应视为已弃用。

连接到数据库

ConnectionString connectionString = new ConnectionString(".....");
MongoClientSettings settings = 
    	//返回'MongoClientSettings.Builder',一个MongoclientSettings的构建器，这样MongoclientSettings可以是不可变的，并且通过链接支持更简单的构造。
    	MongoClientSettings.builder()
    	//返回'MongoClientSettings.Builder',从给定的ConnectionString获取设置，并将它们应用到构建器
        .applyConnectionString(connectionString)
    	//返回'MongoClientSettings',构建一个MongoClientSettings实例。
        .build();
//为使用settings创建一个新连接
MongoClient mongoClient = MongoClients.create(settings);
//获取给定数据库名称的 Mongo 数据库实例。
MongoDatabase database = mongoClient.getDatabase("test");

创建集合

database.createCollection("tutorial");

使用集合

MongoCollection<Document> collection = database.getCollection("TEST");

创建文档

Document document = new Document("title","MongoDBTest")
                .append("time","2022,02,22")
                .append("author","me");

插入文档

collection.insertOne(document);

插入多条文档


Document document1 = new Document("title","title1")
    .append("time","2021,01,11")
    .append("author","me");
Document document2 = new Document("title","title2")
    .append("time","2019,09,19")
    .append("author","you?");
List <Document> list = new ArrayList<>();
list.add(document1);
list.add(document2);
collection.insertMany(list);

查询文档

// 获取 iterable 对象
FindIterable<Document> iterables = collection.find();
// 获取迭代器
Iterator iterator = iterables.iterator();
while(iterator.hasNext()){
    System.out.println(iterator.next());
}

更新文档

collection.updateOne(Filters.eq("title","title1"), Updates.set("title","TITLE1"));
//如果更新的键已存在则更新，不存在则插入新的键值对。

删除文档

collection.deleteOne(Filters.eq("title","TITLE1"));

删除集合

collection.drop();
