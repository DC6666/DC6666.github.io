前缀和与差分

前缀和

给定n个数,m次查询操作，每次求出[l,r]内所有数字的和(0<n,m<100000)

如果每次查询l->r间所以数字的和都遍历一次l->r,当数据范围不大，但是数据量打的时候就会花费很多时间。所以,可以让第i个数的值为前i个数的和，前r个数的值减去前l-1个数的值就等于l->r所有数的值。(如果所有数的值加起来都不会爆掉的话🐶)

#include <iostream>
using namespace std;
int ans[100005];
int main() {
	int n, m, t;
	cin >> n >> m;
	for (int i = 1; i <= n; i++) {
		cin >> t;
		//第i个数的值为前i个数的和
		ans[i] = ans[i - 1] + t;
	}
	for (int i = 0; i < m; i++) {
		int l, r;
		cin >> l >> r;
		//前r个数的值减去前l-1个数的值就等于l->r所有数的值。
		cout << ans[r] - ans[l - 1] << endl;
	}
}

差分

对于n个数,执行k次操作,要将其中 i->j 个数同时加k (i,j<=n),如果使用暴力求解对于每个数都加上相应的值,如果使用暴力求解，时间复杂度会达到k*(j-i).

所以采用差分思想，每个数的当前值为其实际值相对于前一个数实际值的差，当对于i->j个数同时+m时，只需要对第i个数的值+m,对第j+1个数的值-m，对于i->j之间的数来说,所有数的值都加了m。(m为正数或负数)

题目链接：P2367 语文成绩 - 洛谷

#include <iostream>
#include <algorithm>
using namespace std;
int ans[5000005];
int main() {
	int n, p, t{ 0 };
	cin >> n >> p;
	for (int i = 1; i <= n; i++) {
        //ans[i]减去上一个数的值
		ans[i] = -t;
		cin >> t;
        //ans[i]再加上实际的值,最终为ans[i]相对于ans[i-1]的差值
		ans[i] += t;
	}
	for (int i = 0; i < p; i++) {
		int x, y, z;
		cin >> x >> y >> z;
        //ans[x]的值加z,ans[y+1]的值减去z,相当于x到y所有数的值都再原本的基础加m,y之后的数不变
		ans[x] += z;
		ans[y + 1] -= z;
	}
	int minAns = 0x7f7f7f7f;
	int realValue{ 0 };
	for (int i = 1; i <= n; i++) {
        //第i个数的真实值等于前一个数的真值+第i个数的实际值。
		realValue += ans[i];
		minAns = min(minAns, realValue);
	}
	cout << minAns;
}

分治与贪心

分治

分治，字面上的解释是“分而治之”，就是把一个复杂的问题分成两个或更多的相同或相似的子问题，再把子问题分成更小的子问题……直到最后子问题可以简单的直接求解，原问题的解即子问题的解的合并。

分治的步骤：

  • 分解（Divide）：将原问题分解为若干子问题，这些子问题都是原问题规模较小的实例（小问题容易解决）。
  • 解决（Conquer）：递归地求解各子问题。如果子问题规模足够小，则直接求解。（子问题的解可以合并为该问题的解）
  • 合并（Combine）：将所有子问题的解合并为原问题的解（各个子问题是相互独立的）

二分查找法当前值

在一组有序的数据中找到目标值(的位置)。

int bsearch(int* a, int x, int y, int v)
{
    while (x<y)// 注意：不要写成“x<=y”，否则会死循环！
    {
        int mid=x+(y-x)/2;
        //找到相关的值返回
        if (v==a[mid]) 
            return mid;
        //如果中间数的值大于要找的值,从左半边找。
        else if (v<a[mid])
            y=mid-1;
        //如果中间数的值小于要找的值,从右半边找。
        else
            x=mid+1; // 注意：不要忘记加1！
    }
    return -1;// 找不到
}

二分查找求上界

在一组有序的数据中找到比目标值大的数中的最小值。

int upper_bound(int *a, int x, int y, int v)
{
    //当x==y时,说明该值是最后一个大于v的
    while (x < y)
    {
        int mid = x + (y - x) / 2;
        //如果v>=a[mid],找比v的上界值就从mid的右边找。
        if (v >= a[mid])
            x = mid + 1;
        //否则从mid的左边找
        else
            y = mid;
    }
    return x;
}

二分查找求下界

在一组有序的数据中找到比目标值下的数中的最大值。

int lower_bound(int *a, int x, int y, int v)
{
    while (x < y)
    {
        int mid = x + (y - x) / 2;
        //与查找上界正好相反。
        if (v > a[mid])
            x = mid + 1;
        else
            y = mid;
    }
    return y;
}
